# TAMAGURU OS

Tamaguru — это не просто операционная система. Это живое цифровое существо.

## Как запустить:
1. Установи Rust: https://rustup.rs
2. git clone https://github.com/YOUR_USERNAME/tamaguru-os.git
3. cd tamaguru-os
4. cargo run
